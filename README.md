# Cybersecurity Portfolio - Mohammed Saber

A professional, dark-themed portfolio website featuring technical expertise, projects, and certifications.

## 🚀 Live Demo
Once deployed, view it at: `https://sabero01.github.io/Portfolio/`

## 🛠️ Features
- **Dark Mode Aesthetic**: High-contrast, technical design inspired by the security community.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile.
- **Interactive Elements**: Glitch effects, blinking terminal cursor, and hover states.
- **Sectioned Content**: Clear divisions for About, Experience, Projects, and Certifications.

## 📦 Tech Stack
- HTML5
- CSS3 (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter, Fira Code)

## 🌐 Deployment to GitHub Pages
1. Go to your repository on GitHub.
2. Click **Settings** > **Pages**.
3. Under **Branch**, select `main` and `/ (root)`.
4. Click **Save**.
5. Your site will be live in a few minutes!
